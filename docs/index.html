🏁 Getting Started

Kick off with a simple setup:

            static void Main()
            {
                var scheduler = new SchedulerEngine(() =>
                {
                    Console.WriteLine("Scheduled task executed!");
                });

                scheduler.Start();
            }

⚙️ XML Configuration

Add this to your App.config or Web.config:

<configuration>
  <configSections>
    <section name="schedulerSettings" type="AutomateCore.Configuration.SchedulerConfigSection, AutomateCore" requirePermission="false" />
  </configSections>

  <schedulerSettings>
    <add scheduleType="Hourly" enabled="true" hour="0" minute="0" />
  </schedulerSettings>
</configuration>

🗓️ Schedule Attributes

Attribute Description
scheduleTypeImmediate, Daily, Weekly, Monthly, etc.
enabledtrue / false
hourHour value
minuteMinute value
dayOfWeekApplicable for Weekly
dayOfMonthApplicable for Monthly

🧠 Supported Schedule Types

  • Immediate — Runs once on service start
  • EveryXMinutes — Custom intervals like 5, 15, 30 mins
  • EveryXHours — Run every 2, 4, 6 hours
  • Hourly — Once per hour at a given minute
  • Daily, Weekly, Monthly — Classic patterns
  • TwiceDaily, Weekends, Quarterly — Extended options

🧑‍💻 Lifecycle Callbacks

private ISchedulerEngine _scheduler;

void RunTask() => Console.WriteLine("Running scheduled task...");

void NotifyServiceStarted(DateTime t) => Console.WriteLine($"✅ Started at {t}");
void NotifyServiceCompleted(DateTime t) => Console.WriteLine($"✅ Completed at {t}");
void NotifyServiceSkipped(string reason, DateTime t) => Console.WriteLine($"⚠️ Skipped — {reason}");
void NotifyServiceFailed(Exception ex, DateTime t) => Console.WriteLine($"❌ Failed — {ex.Message}");

_scheduler = new SchedulerEngine(
    RunTask,
    onTaskStarted: NotifyServiceStarted,
    onTaskCompleted: NotifyServiceCompleted,
    onTaskSkipped: NotifyServiceSkipped,
    onTaskFailed: NotifyServiceFailed
);
