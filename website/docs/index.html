Frequently Asked Questions

How do I handle errors in my tasks?

The SchedulerEngine automatically logs all exceptions. For custom error handling, wrap your task logic in try-catch blocks. You can also use the WithRetry() extension method to add retry behavior.

Can I schedule a task to run every 15 minutes?

Yes! Use the Interval schedule type with a TimeSpan of 15 minutes. Alternatively, configure an hourly schedule with specific minute marks (0, 15, 30, 45).

Is there a way to monitor running tasks?

AutomateCore integrates with ASP.NET Core's health checks. Implement IHealthCheck to expose task status. You can also subscribe to engine events for real-time monitoring.

How do I run tasks in a cluster environment?

For clustered deployments, use the DistributedLock feature to ensure tasks only run on one node. Configure your lock provider in the scheduler options.
